모든 방위는 기본적으로 **길방위(is_auspicious = True)**를 기본값으로 가지며, 아래 서술된 여러 가지 **흉방위(凶方位) 조건**을 하나라도 만족할 경우 해당 방위는 흉방위(is_auspicious = False)로 전환됩니다.

## 1. 흉방위(凶方位) 판별 조건 (7대 주요 흉살)

### ① 五黄殺 (고오사츠 / 오황살)
*   **조건**: 해당 방위에 위치한 별(Star)이 **5 (五黄土星, 오황토성)**인 경우.
*   **부여 마크**: `five_yellow`
### ② 暗剣殺 (안켄사츠 / 암검살)
*   **조건**: 오황살(5번 별)의 **정반대 편에 위치한 방위**.
*   **예외**: 만약 중앙(Center)에 5번 별이 위치해 있다면(즉, 오황중궁), 암검살은 모든 방위에 존재하지 않습니다.
*   **부여 마크**: [dark_sword]
### ③ 本命殺 (혼메이사츠 / 본명살)
*   **조건**: 사용자의 **본명성(main_star)**과 동일한 번호를 가진 별이 위치한 방위.
*   **부여 마크**: `main_star`
### ④ 月命殺 (겟메이사츠 / 월명살)
*   **조건**: 사용자의 **월명성(month_star)**과 동일한 번호를 가진 별이 위치한 방위.
*   **부여 마크**: `month_star`
### ⑤ 水火殺 (스이카사츠 / 수화살)
*   **조건**: 방위의 별이 1(일백수성) 또는 9(구자화성)이면서, 현재 구성반의 남쪽(South)에 1이 있거나 북쪽(North)에 9가 있는 경우.
    *(설명: 본래 물(1)의 자리인 북쪽에 불(9)이 오거나, 불(9)의 자리인 남쪽에 물(1)이 올 때 발생하는 흉운을 의미합니다.)*
*   **부여 마크**: `water_fire`
### ⑥ 破 (야부레 / 파) - 年破, 月破, 日破 등
*   **조건**: 기준이 되는 시간(연/월/일/시)의 **간지(干支)의 정반대 편 방위**.
    *   목표 간지의 '지(支)'를 추출 (예: 甲寅 -> 寅)
    *   마주 보는 지(支)를 산출: 子 <-> 午, 丑/寅 <-> 未/申, 卯 <-> 酉, 辰/巳 <-> 戌/亥
    *   반대편 지(支)가 속한 방위를 흉방위로 지정:
        *   子(북), 丑/寅(북동), 卯(동), 辰/巳(남동), 午(남), 未/申(남서), 酉(서), 戌/亥(북서)
*   **부여 마크**: [opposite_zodiac]
### ⑦ 凶方星 (상성 배점 기반 흉방)
*   **조건**: 본명성과 해당 방위에 위치한 별 간의 **구성을 조합한 상성 수준(Compatibility Level)**이 `BAD`(나쁨)인 경우.
*   `StarCompatibilityMatrix`를 통해 관계를 조회합니다(오행의 상생상극 원리 적용).
*   **부여 마크**: `compatibility_matrix`

## 2. 사후(Post-loop) 판별 조건 (的殺 / 적살)
위 7가지 조건을 지도 전체 방위에 반복문으로 우선 확인한 후, 본명/월명성이 위치한 포지션을 단서로 반대편을 추가로 흉격 처리합니다.
### ⑧ 本命的殺 (혼메이테키사츠 / 본명적살)
*   **조건**: ③ 본명살(본명성이 위치한 방위)의 **정반대(Opposite) 방위**.
*   **부여 마크**: `main_star_opposite`
### ⑨ 月命的殺 (겟메이테키사츠 / 월명적살)
*   **조건**: ④ 월명살(월명성이 위치한 방위)의 **정반대(Opposite) 방위**.
*   **부여 마크**: `month_star_opposite`
