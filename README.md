# LumiNine (루미나인)

> 구성기학의 운명적 방향과 천연석의 에너지를 동기화하는 맞춤형 파워스톤 큐레이션 서비스

## 프로젝트 개요
나의 본명성(本命星)과 매월 변하는 우주의 기운(길흉방위)을 분석하여, 현재 나에게 가장 필요한 에너지를 채워주는 3가지 파워스톤(기본석, 월운석, 호신석)을 개인 맞춤형으로 큐레이션합니다.

## 핵심 가치
- **운명적 동기화**: 구성기학 로직을 기반으로 나와 우주의 주파수를 맞춤
- **프리미엄 큐레이션**: 천연석 고유의 오행(五行) 에너지를 정밀하게 매칭
- **현대적 오컬트**: 전통의 명리학을 현대적이고 우아한 라이프스타일 굿즈로 재해석
